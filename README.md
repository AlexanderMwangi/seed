# Sails.js Seed (Sample App)

A sample [Sails](http://sailsjs.com) application, generated using:

```
npm install -g sails@balderdashy/sails#87624e7aa9aedddb3c8ee75decd37e4ff2a4ee83
```


And then:

```
sails new seed
```
